# daredevil
Magnificent Sublime plugin to outline the current line and highlight the gutter (specially useful when cycling through search results).

![Daredevil screenshot](https://github.com/imagentleman/daredevil/blob/gh-pages/daredevil-screenshot.png)

# Installation

1. Download the lastest version of daredevil from <a href="https://github.com/imagentleman/daredevil/raw/master/daredevil.py" download>here</a>.
2. Go to Sublime's user packages (`Preferences > Browse Packages > user`) and place the plugin there.
3. Success.
